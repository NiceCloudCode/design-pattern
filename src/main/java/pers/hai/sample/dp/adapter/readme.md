## Adapter Design Pattern.

> 适配器模式：将一个类的接口转换成客户希望的另外一个接口。Adapter 模式使得原本由于接口不兼容而不能一起工作的那些类可以一起工作。

- entrust -- 基于委托实现的适配器模式
- inherit -- 基于继承实现的适配器模式
